# OT/ICS Security Glossary

| Term | Definition |
|---|---|
| **4-20mA** | Analog signal standard used in process industry for sensor data (4mA=0%, 20mA=100%) |
| **ACL** | Access Control List — firewall rules defining permitted traffic |
| **BPCS** | Basic Process Control System — the DCS/PLC system controlling normal operations |
| **CIP** | Common Industrial Protocol — application layer used by EtherNet/IP, DeviceNet, ControlNet |
| **CODESYS** | IEC 61131-3 PLC programming environment used by many vendors |
| **Conduit** | IEC 62443 term for a communication path between two security zones |
| **DCS** | Distributed Control System — continuous process control across multiple controllers |
| **DCOM** | Distributed Component Object Model — Windows middleware used by OPC Classic |
| **DPI** | Deep Packet Inspection — application-layer firewall analysis |
| **ESD** | Emergency Shutdown — automatic safe-state trigger from SIS |
| **EWS** | Engineering Workstation — computer used to program PLCs/DCS |
| **GICSP** | Global Industrial Cyber Security Professional — primary ICS security certification |
| **HART** | Highway Addressable Remote Transducer — digital communication over 4-20mA wiring |
| **HMI** | Human-Machine Interface — operator screen showing process status |
| **Historian** | Time-series database storing process data (OSIsoft PI, Wonderware Historian) |
| **ICS** | Industrial Control System — umbrella term for SCADA, DCS, PLC-based systems |
| **IED** | Intelligent Electronic Device — smart relay or substation device in power grid |
| **IACS** | Industrial Automation and Control System — term used in IEC 62443 |
| **ISA** | International Society of Automation — standards body for industrial automation |
| **Ladder Logic** | PLC programming language resembling electrical relay diagrams |
| **MES** | Manufacturing Execution System — production management between ERP and control |
| **MTTD** | Mean Time To Detect — average time to discover a breach |
| **MTTR** | Mean Time To Respond — average time to contain a breach |
| **OPC** | OLE for Process Control — middleware for vendor-neutral HMI↔PLC communication |
| **OT** | Operational Technology — hardware/software controlling physical processes |
| **P&ID** | Piping and Instrumentation Diagram — process engineering layout document |
| **PAM** | Privileged Access Management — controls for high-privilege accounts |
| **PLC** | Programmable Logic Controller — ruggedized computer executing control logic |
| **PROFIBUS** | Serial fieldbus protocol (Siemens) for field device communication |
| **PROFINET** | Industrial Ethernet protocol (Siemens) for PLC↔I/O communication |
| **RTU** | Remote Terminal Unit — remote field data collection device |
| **SCADA** | Supervisory Control and Data Acquisition — centralized monitoring and control |
| **SIL** | Safety Integrity Level — SIS reliability rating (SIL 1 lowest, SIL 4 highest) |
| **SIS** | Safety Instrumented System — independent emergency shutdown layer |
| **SPAN** | Switch Port Analyzer — mirror port that copies traffic for passive monitoring |
| **Tag** | Named reference to a process variable (e.g., TIC-101 = temperature controller) |
| **TIA Portal** | Siemens engineering software for programming S7-1200/1500 PLCs |
| **Zone** | IEC 62443 term for a logical grouping of OT assets with similar security requirements |
