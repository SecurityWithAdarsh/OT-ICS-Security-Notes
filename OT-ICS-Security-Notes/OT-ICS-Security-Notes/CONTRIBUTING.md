# Contributing to OT-ICS-Security-Notes

Thank you for your interest in contributing! This repository aims to be the most practical, field-informed OT/ICS security knowledge base available.

## What We Welcome
- Corrections to technical inaccuracies
- New protocol coverage (Fieldbus, IEC 61850, BACnet, etc.)
- Additional case studies (post-2022 incidents)
- GICSP practice questions
- Tool guides and lab walkthroughs
- Real-world field observations (sanitized — no proprietary/confidential info)

## How to Contribute
1. Fork this repository
2. Create a feature branch: `git checkout -b feature/add-iec61850-notes`
3. Make your changes following the existing format
4. Submit a pull request with a clear description

## Standards for Contributions
- No proprietary, classified, or confidential operational data
- All technical claims should be verifiable from public sources
- Follow the existing Markdown format
- Include Interview Q&A section if adding a new module
- Credit sources where appropriate

## Reporting Issues
Open a GitHub issue for factual errors, outdated information, or missing topics.
